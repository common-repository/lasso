Enables shortcode to embed ToneDen's widget in WordPress blogs.

== Description ==

ToneDen is the easiest way to turn anonymous website visitors into real customers.
This plugin makes installation of the widget a simple one-step process.

== Installation ==

1. Download lasso.zip from WordPress, and unzip it.
1. Upload the `lasso/trunk` directory from the unzipped folder to your site's `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a ToneDen account at www.toneden.io/signup, then visit
www.toneden.io/lasso/campaign and set up your Lasso embed.
1. Go to www.toneden.io/lasso/install/wordpress and copy the displayed user ID.
1. Embed Lasso with shortcode: `[lasso user_id=<your user ID>]`.
